# MealDB
iOS App to list meal recipes and allow the user to favorite recipes

## Getting Started

Clone the project and open the Xcode Project

### Prerequisites

There are no third party libraries or external dependencies required.

## Built With

  - UIKit
  - Xcode 13
  - Swift 5
  - Async/Await
